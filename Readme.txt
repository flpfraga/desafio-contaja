Desafio projeto Contajá

Esse sistema foi desenvolvido como parte de um desafio de desenvolvimento do front-end de uma aplicação.
Sua funcionalidade é uma loja de quadrinhos, onde o usuário pode visualizar uma listagem com diversos exemplares, 
buscar exemplares através do título, ver detalhes sobre o quadrinho, optar por comprar ou adicionar ao carrinho e
por fim, acessar seu carrinho de comprar e verificar os exemplares que selecionou para comprar. 


Para seu desenvolvimento foi utilizado o framework React.js, que se baseia na construção de componentes menores
para juntos construirem grande componentes. Utiliza como linguagem básico tanto o JavaScript quanto o TypeScript.
O atual projeto foi desenvolvido com a linguagem JavaScript.

A estilizaçao foi feita através do Styled Components, que é uma biblioteca que permite escrever código CSS dentro 
do JavaScript. Ela garantiu à aplicação maior independencia e organização na estilizaçao dos componentes.

Os exemplares dos quadrinho são obtidos através da API pública disponível através do link https://developer.marvel.com
Na construção da api foi utilizado o axios, o qual fez a busca dos dados na API e entregou para o sistema.

Além dos já citados, também foi utilizado biblioteca de icones, React Icons.
